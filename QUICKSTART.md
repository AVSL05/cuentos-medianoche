# Quick Start Guide - Cuentos de Medianoche

## 🚀 Get Started in 2 Minutes

### First Time?

1. **Visit the app:** https://cuentos-medianoche.vercel.app/
2. **Read the welcome message** - Choose your role:
   - **🌙 La Luna** (Listener) - Listen to stories
   - **☀️ El Sol** (Creator) - Upload stories
3. **Enjoy!**

---

## 👂 For La Luna (Listeners)

### Playing Your First Story

```
1. Open app
2. Choose "La Luna" on first screen
3. See list of uploaded stories
4. Click any story to play
```

### Player Controls

| Control | Action |
|---------|--------|
| **Story Card** | Click to play/pause |
| **▶ / ⏸** | Play or pause |
| **Seek Bar** | Drag to jump to any time |
| **🔊** | Adjust volume |
| **15min / 30min / 60min** | Set sleep timer |

### Sleep Timer

```
While playing:
1. Click desired duration (15, 30, or 60 minutes)
2. Blue timer bar appears
3. Audio stops automatically when time runs out
4. Click "Cancelar" to stop timer early
```

### Tips

- ✓ Timer keeps playing even if you minimize browser
- ✓ Use sleep timer to prevent phone from staying on
- ✓ Close browser anytime - no data lost
- ✓ Refresh to reload story list
- ✓ Your role is saved - no need to select again

### Troubleshooting

**Stories won't load?**
- Refresh the page (Ctrl+R or Cmd+R)
- Check internet connection
- Try a different browser

**Audio won't play?**
- Click play button again
- Try a different story
- Check browser volume isn't muted
- Close and reopen browser

**Sleep timer not working?**
- Refresh page
- Ensure audio is actually playing
- Check browser console (F12) for errors

## 💿 For El Sol (Creators)

### Upload Your First Story

```
1. Open app
2. Choose "El Sol" on first screen
3. Click "+ Agregar cuento"
4. Click the upload area to select file
5. System auto-fills title (you can edit)
6. Click "☁️ Guardar en la nube"
7. Wait for upload to complete
```

### Upload Process

**Step 1: Select File**
```
- Click drop area
- Choose MP3, M4A, WAV, OGG, or AAC file
- File size shown in MB
```

**Step 2: Edit Title (Optional)**
```
- Auto-filled from filename
- Remove file extension automatically
- Can customize: "My Story" → "Mi Historia"
```

**Step 3: Upload**
```
- Click green button
- Progress bar shows:
  - "Subiendo audio..." = Uploading to cloud (0-80%)
  - "Guardando en la nube..." = Saving info (80-100%)
  - "¡Listo!" = Complete!
```

**Step 4: Done!**
```
- New story appears at top of list
- Panel closes automatically
- Story ready to listen
```

### Manage Stories

**Delete a Story:**
```
1. Find story in list
2. Click × button on right
3. Story removed immediately
```

**Replace a Story:**
```
1. Delete old version
2. Upload new version
3. New upload appears at top
```

### File Requirements

✓ **Accepted Formats:**
- MP3 (most common)
- M4A (iPhone recordings)
- WAV (high quality)
- OGG (open format)
- AAC (iTunes format)

✓ **Size Limits:**
- Recommended: Under 50MB
- Maximum: ~100MB
- Larger files take longer to upload

⚠️ **Not Supported:**
- WMA, FLAC, DSD
- Video files (audio only)
- Compressed archives

### Tips for Good Uploads

1. **Filename Becomes Title**
   ```
   Good: "Mi_Primera_Historia"  →  "Mi Primera Historia"
   Avoid: "story_v3_final_FINAL_real.mp3"
   ```

2. **Test Before Uploading**
   - Verify audio quality
   - Check no clicks/pops
   - Ensure proper volume

3. **Name Descriptively**
   ```
   ✓ "Aventura en el Bosque Encantado"
   ✓ "Relajación - Meditación Guiada"
   ✗ "Audio1", "Story", "Final"
   ```

4. **Upload Time Depends On:**
   - File size
   - Your internet speed
   - Server load
   - Audio length

### Troubleshooting

**Upload button disabled?**
- Select a file first
- Wait for current upload to finish

**Upload fails?**
- Check internet connection
- Try smaller file
- Refresh and try again
- Check browser supports HTML5

**Progress bar stuck?**
- Wait up to 2 minutes
- If still stuck, refresh page
- Check browser console (F12)

**Title not auto-filling?**
- Manually enter title
- Special characters may cause issues
- Keep title under 50 characters

## 🔐 Privacy & Data

### What's Stored

**On Your Device (localStorage):**
- Your chosen role (Luna/Sol)
- Whether you've seen welcome message

**On Cloud (Cloudinary):**
- Audio files you upload
- Stored for streaming

**On Cloud (JSONBin):**
- Story titles and metadata
- Upload dates
- Story URLs

### What's NOT Stored
- ✓ Listening history
- ✓ Play counts
- ✓ Personal information
- ✓ Passwords
- ✓ Cookies or tracking

### Can Others See?

**Story List is PUBLIC:**
- Anyone with the URL can see all stories
- Anyone can listen to all stories
- No password protection
- Role doesn't restrict viewing (just uploading)

**To Keep Private:**
- Don't share the URL
- Use private network only
- This is a personal app, not a social platform

## 🎧 Best Practices

### For La Luna (Listeners)

✅ **Do:**
- Set sleep timer before falling asleep
- Use quality headphones/speakers
- Keep device nearby but safe
- Have backup story if one loads slowly

❌ **Don't:**
- Leave volume too loud (hearing safety)
- Cover audio jack (prevent overheating)
- Drop device in bed (damage risk)
- Leave on maximum brightness

### For El Sol (Creators)

✅ **Do:**
- Record in quiet room (no background noise)
- Speak clearly and at comfortable pace
- Test audio quality before uploading
- Write good story titles
- Use consistent volume levels

❌ **Don't:**
- Upload copyrighted material
- Record over device speaker (echo)
- Compress audio excessively
- Use extremely long files (>2 hours)
- Upload inappropriate content

## 📚 Common Questions

### Q: Can I download stories?
**A:** Not directly through the app. Stories are streamed from cloud. Contact El Sol for original files.

### Q: Can I share the app with others?
**A:** Yes! Anyone can visit the URL and listen to stories. Share the URL: https://cuentos-medianoche.vercel.app/

### Q: Can I edit uploaded stories?
**A:** No. Delete and re-upload to change title or content.

### Q: Is there backup/backup restore?
**A:** Stories stored in cloud (Cloudinary + JSONBin). No manual backup needed.

### Q: Can I use two devices?
**A:** Yes! Story list syncs across all devices automatically.

### Q: What if I forget my role?
**A:** Click "cambiar" link to select role again.

### Q: Can I hide stories from La Luna?
**A:** No. All stories visible to anyone with URL. Not designed for sharing restrictions.

### Q: Does it work offline?
**A:** No. Needs internet for upload, playback, and storage access.

### Q: Can multiple people upload?
**A:** Yes, but anyone with URL can upload (no access control). Use private network only.

### Q: Why is audio quality lower?
**A:** Check original file quality. Cloudinary may compress based on upload preset.

### Q: Can I use this for podcasts?
**A:** Yes, technically. But designed for personal use, not distribution.

## 🆘 Emergency Troubleshooting

### Checklist Before Contacting Support

```
□ Refresh page (Ctrl+Shift+R)
□ Close and reopen browser
□ Check internet connection
□ Try different browser
□ Clear browser cache (Settings → Clear browsing data)
□ Disable browser extensions
□ Try private/incognito mode
□ Check Cloudinary status (cloudinary.com/status)
□ Check JSONBin status (jsonbin.org)
```

### Can't Upload?

```
1. Verify file format (MP3, M4A, WAV, OGG, AAC)
2. Check file size (<100MB)
3. Verify you're El Sol
4. Check internet speed
5. Try smaller file
6. Try different browser
7. Wait 5 minutes and retry
```

### Can't Play Audio?

```
1. Refresh page
2. Click story again
3. Try different story
4. Check volume (not muted)
5. Try different browser
6. Check internet connection
7. Restart browser
```

### Stories Won't Load?

```
1. Refresh page
2. Wait 30 seconds
3. Check internet connection
4. Try different network (WiFi vs cellular)
5. Clear browser cache
6. Try private/incognito mode
```
